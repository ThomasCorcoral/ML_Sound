Training 3

TKinter

1 - Introduction

Tkinter est la bibliothèque graphique que nous utiliserons pour notre programme. Je vais ici vous présenter les principaux éléments que nous utiliserons. La suite sera présentée sous une forme de TP. Si vous souhaitez le faire en même temps ouvrez pycharm, créez un nouveau fichier et une fois sur ce nouveau fichier pour le lancer faites « clique droit > run … » Avant tout nous allons réaliser les imports
# Cette ligne signifie que l’on souhaite importer l’entièreté des fonctions / classes de tkinter
from tkinter import *

2 - Préparation

Pour commencer j’ai tendance à créer une fenêtre de taille fixe avec ses dimensions renseignées dans des variables globales. Disons que notre fenêtre va faire 1500x800. On enregistre donc ces valeurs :
WIDTH = 1500
HEIGHT = 800

Ensuite on va préparer notre main :
if __name__ == "__main__":

3 - Début avec Tkinter

Maintenant il va nous falloir une fenêtre. Nous allons l’initialiser et l’afficher (dans notre main bien sur) :
window = Tk()
window.mainloop()

Nous avons désormais une fenêtre qui s’appelle window. Maintenant nous allons personnaliser notre fenêtre en lui donnait un titre, ses dimensions et en verrouillant ses dimensions :
window.title("Tuto Tkinter") # titre
window.geometry(str(WIDTH) + "x" + str(HEIGHT)) # taille initiale
window.minsize(WIDTH, HEIGHT) # Impossible de réduire la fenêtre
window.maxsize(WIDTH, HEIGHT) # Impossible d’agrandir la fenêtre

4 - Les Canvas

Désormais il va falloir placer des éléments sur cette fenêtre, nous allons commencer par un Canvas. Un canvas est une zone délimitée qui va par exemple nous permettre de dessiner dessus ou d’y afficher une image. Notre premier canvas aura une taille de 300x300 avec un fond gris
my_can = Canvas(window, width=300, height=300, bg="grey")

5 - Le placement

Vous pourrez constater que ce canvas n’apparaît pas. En effet avec tkinter dès que l’on souhaite afficher un élément il faut le préciser. Trois fonctions existent, pack(), grid() et place(). Le premier est assez violent car il permet uniquement de placer un élément qu’à gauche, droite, haut, bas. Le second se base sur un système de grille et le dernier permet de placer un élément à des coordonnées précises. Personnellement j’aime bien le place() pour une fenêtre de taille fixe.
my_can.place(x=50, y=50)

6 - Les images

Pour les images il est également possible d'utiliser la bibliothèque Pillow (PIL). Maintenant rajoutons une image à notre canvas que l’on va redimensionner en la rendant 6 fois plus petites
my_img = PhotoImage(file=’image.png').subsample(6)
my_can.create_image(0, 0, anchor=NW, image=my_img, tags="IMG ")

7 - Les Boutons

Notre fenêtre est désormais décorée mais elle reste inutile. Nous allons donc voir comment créer un bouton :
my_but = Button(window, text="OPEN", font=("Courrier", 21), fg=’black')
my_but.pack(side=RIGHT)

webbrowser.open_new(link)

my_but= Button(window, text="OPEN", font=("Courrier", 21), fg='white', command=lambda: open_website (“https://projet.xnh.fr/”))
