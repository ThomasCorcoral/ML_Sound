Training 2

Bases en Python

1 - API et bibliothèques

Pour utiliser des API et autres bibliothèques il faut faire un import. Il est possible de donner un nom spécifique à l’import avec un ‘as’ par exemple
Import tkinter as tk

Si lors de votre compilation une erreur apparait à ce niveau-là c’est surement que votre environnement virtuel ne connait pas cette bibliothèque. Il faudra donc l’installer. Il suffit d’ouvrir un terminal local dans pycharm et d’executer une commande du type

pip install tkinter

2 - Les commentaires

Pour commenter une simple ligne il faut utiliser le caractère #
Pour commenter un bloc de code / texte il faut utiliser les balises : """

3 - Les variables

En python les variables n’ont pas de type. Il suffit de faire un x = 5 et le typage se fera automatiquement. C’est la même chose pour les paramètres des fonctions. Pour définir une variable globale il suffit de la mettre hors de toute fonction. Si vous souhaitez rendre une variable interne à une fonction globale (ce qui est un peu laid) il faut écrire en tête de cette fonction global x

4 - Les fonctions

Comme les variables les fonctions n’ont pas de type, elles sont donc par nature toutes des procédures. Cependant vous pouvez renvoyer ce que vous souhaitez. Pour définir une fonction il suffit d’écrire def foo(x): Il est également possible de retourner plusieurs valeurs (génial !) pour cela il suffit de return les différentes valeurs return a, b Il faudra donc faire attention lors de la récupération du résultat. Pour créer le main il faut suivre la syntaxe suivante if __name__ == "__main__":

5 - Les conditions

En python contrairement au c et c++ les { n’ont pas d’utilité dans la délimitation des fonctions ou des conditions. Ici ce sont les alinéas qui vont jouer ce rôle. Les conditions sont les mêmes que dans les autres langages. Veilliez bien à mettre les ‘:’ après votre condition et également l’alinéa de début

6 - Les classes

En python les classes sont semblables à celles de java. Elles possèdent des variables internes qui sont stockées tant que la classe créée n’est pas supprimée. Elle peut posséder des fonctions internes. Pour accéder à ses variables dans une fonction interne il faut utiliser le mot ‘self’ par exemple pour accéder à la variable x self.x La fonction d’initialisation doit posséder la syntaxe suivante : def __init__(self, …): Il est préférable d’initialiser toutes les valeurs utiles à la classe dans cette fonction.
