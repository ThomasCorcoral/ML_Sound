CSV path : Selectionnez votre fichier contenant les informations sur votre dataset (.csv)
           Faites attention au contenu de votre fichier. Il doit contenir 4 informations :
                - name : Le nom du fichier
                - folder : Le dossier du fichier dans votre dossier data, si aucun mettez '.'
                - class : La classe de votre enregistrement (exemple : 0,1,2)
                - class_name : Les noms de la classe correspondante (exemple: Pipistrelle, Rhinolophe etc)
