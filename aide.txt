Data path : Appuyer sur ce bouton fera apparaitre une fenetre. Depuis cette fenetre, selectionnez le dossier contenant
votre "dataset" (Dossier compose de fichiers .wav).

CSV path : Appuyer sur ce bouton fera apparaitre une fenetre. Depuis cette fenetre, selectionnez votre fichier contenant
les informations sur votre dataset (Un fichier tableur type Excel .csv).
           Faites bien attention au contenu de votre fichier. Il doit contenir les 4 informations formatees dans cet ordre :
                - name : Le nom du fichier sonore.
                - folder : Le dossier dans lequel se trouve le fichier dans votre dossier "data", s'il n'y en a aucun, mettez '.'
                - class : La classe de votre enregistrement, c'est-a-dire la separation entre dossiers / noms de
                chauves-souris dans notre cas (exemple : 0, 1, 2).
                - class_name : Les noms de la classe correspondante (exemple: Pipistrelle, Rhinolophe, etc...).

Format data : Prepare les donnees indiquees par les paths, et en extrait des fichiers .npy en fonction des options selectionnees.

Run Train : Lancez l'entrainement du modele, ce qui va permettre d'obtenir des resultats plus precis en fonction du nombre d'epochs 

Epochs = Repetitions de l'entrainement. Plus d'epochs signifie une meilleure precision, mais aussi un plus grand temps d'entrainement et surtout un risque de surapprentissage. 
Surapprentissage = Une analyse trop precise sur un modele, ce qui signifie que les futures analyses ne pourra pas prevoir correctement les observations dans le futur.

Vous pouvez selectionner le nombre d'epochs manuellement.

Le pourcentage que vous verrez apres est la precision estimee de l'entrainement.

Test path : Chemin vers votre fichier a tester en utilisant l'entrainement precedent. Il doit s'agir d'un fichier sonore .wav

Run test : Permet de lancer la reconnaissance du modele.


Parmis les autres boutons :

Play Test File : Permet de jouer le son indiqué dans Test Path

Enregistrer sous : Permet d'enregistrer le modèle actuel afin de pouvoir revenir dessus sans refaire toutes les manipulations

Generer CSV : Permet de générer le .csv du dataset indiqué de manière automatique. N'oubliez pas de déplacer le .csv hors du fichier local_saves (Car clear après)
ATTENTION : Pour générer le .csv, veuillez bien faire attention à ce que les sons de chaque oiseau / chauve-souris soit dans un fichier ayant pour nom l'espèce de l'animal
(Par exemple, in fichier "data" avec un fichier pour chaque espèce, avec les sons de l'espèce dedans)

Voir audio: Permet de visualiser la représentation graphique du son de Test Path

Voir Spectrogramme : Permet de visualiser le spectrogramme du son de Test Path

Voir MFCC: Permet de visualiser le mfcc du son de Test Path
