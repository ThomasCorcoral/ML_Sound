Training 1

Découverte - Initialisation

1 - Installer python sur votre machine

La suite de l'explication se fera sous Linux (Ubuntu 18.04)
sudo apt update
sudo apt -y update

Vérifiez votre version de python

python3 -v

Si aucun résultat, installez python

2 - Installer pip

sudo apt install -y python3-pip
sudo apt install build-essential libssl-dev libffi-dev python3-dev

3 - Installer PyCharm

Ouvrez le logiciel GNU "Logiciels Ubuntu"
Cherchez PyCharm
Installez "Pycharm CE"

4 - Configurez un nouveau projet

Cliquez sur nouveau projet
Normalement la sélection est déjà bien faite laissez cocher "New emvironment using Virtualenv"
Cliquez sur create

5 - Prise en main

Prenez en main l'environnement
Faites un Run du main (Normalement il est pré-généré et va écrire un petit message)

6 - Installation des dépendances

Attention long et couteux en bande passante

Tapez les commandes suivantes
pip install --upgrade tensorflow
pip install --upgrade numpy
pip install --upgrade matplotlib
sudo add-apt-repository ppa:graphics-drivers
sudo apt-key adv --fetch-keys http://developer.download.nvidia.com/compute/cuda/repos/ubuntu1804/x86_64/7fa2af80.pub
sudo bash -c 'echo "deb http://developer.download.nvidia.com/compute/cuda/repos/ubuntu1804/x86_64 /" > /etc/apt/sources.list.d/cuda.list'
sudo bash -c 'echo "deb http://developer.download.nvidia.com/compute/machine-learning/repos/ubuntu1804/x86_64 /" > /etc/apt/sources.list.d/cuda_learn.list'
sudo apt update

/!\ La commande suivante déclenche une longue installation qui se compte en Go

sudo apt install cuda-10-1

sudo apt install libcudnn7
sudo apt-get install python3-tk
